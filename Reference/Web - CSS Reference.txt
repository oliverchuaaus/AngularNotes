3 Ways to insert CSS
a. external stylesheet
	<link rel="stylesheet" type="text/css" href="mystyle.css">
b. internal stylesheet
	<head>
	  <style>
	  body {
	    background-color: white;
	  }
	  h1 {
	    color: maroon;
	    margin-left: 40px;
 	  } 
  	  </style>
	</head>
c. inline style
	<h1 style="color:blue;">This is a heading.</h1>
	
Cascade will be from external to internal to inline
so inline overrides the declaration of any same selector.
Make sure external declared before internal for this to work.

POSITIONING
4 Positioning Methods:
a. Static	default positioning, top, bottom, left right properties will not work unless position is any of the three below
b. Fixed	relative to browser window, removed from normal flow	
c. Absolute	relative to first parent that has a position other than static, removed from normal flow
d. Relative	relative to its position, preserved in normal flow, usually used as container blocks for absolute position blocks
e. Sticky	relative then fixed when scrolled

Tips:	
When aligning elements with position, always define margin and padding for the <body> element. This is to avoid visual differences in different browsers.
There is also a problem with IE8 and earlier, when using position. If a container element (in our case <div class="container">) has a specified width, and the !DOCTYPE declaration is missing, IE8 and earlier versions will add a 17px margin on the right side. This seems to be space reserved for a scrollbar. So, always set the !DOCTYPE declaration when using position:	
	

COMBINATOR
4 types of combinators
a. Descendant Selector	e.g. div p	p nested inside div
b. Child Selector	e.g. div > p	p immediately inside div
c. Adjacent Sibling	e.g. div + p	first p outside div
d. General Sibling	e.g. div ~ p	subsequent ps outside div

ATTR SELECTORS
a[target] {
    background-color: yellow;
} - only a elements with target attribute will get style
<a href="a" target="b">
a[target="_blank"]
[title="flower"]  - title attribute containing exactly flower e.g. flower
[title~="flower"] - title attribute containing exactly flower in the list e.g. title="flower plant"
[title*="flower"] - title attribute ending with word flower, can be partial e.g. flowerpower
[title|="flower"] - title attribute starting with word flower e.g. flower-plant
[title^="flower"] - title attribute starting with flower e.g. flower-plant, flowerpower
[title$="flower"] - title attribute ending with word flower

5. Install SQLDeveloper
a. Turn on time component:
Go to Preferences > Type in NLS > Type ‘DD-MON-YYYY HH24:MI:SS’ in Date Format
b. Turn on Line Numbers
Go to Preferences > Type in Line Gutter > Check Show Line Numbers

For Oracle XE:
In SQLDeveloper, 
a. Add new connection named "sys@xe" sys/george# and change role from default to SYSDBA.
b. Turn off password expiry of oraclexe
alter profile default limit password_life_time unlimited;
c. Connect and Change port to 8081 by running 
EXEC DBMS_XDB.SETHTTPPORT(8081);
	1. Run to check if port has been changed: 
		select dbms_xdb.gethttpport as "HTTP-Port", dbms_xdb.getftpport as "FTP-Port" from dual;
	2. Update C:\Work\oraclexe\app\oracle\product\11.2.0\server\Get_Started.url to point to port 8081
	3. Open URL to test.
d. increase processes:
	1. Connect as SYSDBA
	2. Run: alter system set processes=500 scope=spfile;
	3. Restart Oracle XE Service
	4. Run: show parameter processes; to verify processes has been increased to 500
e. Create users
create user sandbox identified by sandbox;
grant dba to sandbox;
create user jpasandbox identified by jpasandbox;
grant dba to jpasandbox;
